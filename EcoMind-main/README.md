# 🚀 EcoMind Smart Infrastructure Platform

## AI-Powered Smart Infrastructure & Mall Intelligence Ecosystem

EcoMind Smart Infrastructure Platform is a next-generation AI-powered smart mall monitoring and infrastructure intelligence ecosystem designed to transform modern commercial infrastructures into intelligent, sustainable, scalable, and data-driven environments.

The platform integrates:

- Real-time infrastructure monitoring
- AI-powered operational analytics
- Sustainability & ESG intelligence
- Occupancy tracking & optimization
- Predictive analytics
- Smart anomaly detection
- Digital twin visualization
- Intelligent automation workflows
- Cloud-native realtime architecture

into a centralized enterprise-grade smart dashboard.

Unlike traditional monitoring systems that rely on fragmented operational tracking and reactive management approaches, EcoMind delivers a unified intelligent ecosystem capable of real-time infrastructure visibility, predictive decision support, and sustainability-driven optimization.

# ⚙️ Installation & Setup

## 📌 Prerequisites

Ensure the following tools are installed:

- Node.js (v18+ recommended)
- npm
- Git
- VS Code or preferred editor

---

## Clone Repository

```bash
git clone https://github.com/your-username/EcoMind-Smart-Infrastructure-Platform.git
```

---

## Navigate to Project

```bash
cd EcoMind-Smart-Infrastructure-Platform
```

---

## Install Dependencies

```bash
npm install
```

This installs:
- React ecosystem packages
- Vite dependencies
- Tailwind CSS
- React Router
- TanStack Query
- Supabase integrations
- Dashboard visualization libraries
- Cloud connectivity modules

---

## Run Development Server

```bash
npm run dev
```

The application runs locally on:

```text
http://localhost:8080
```
